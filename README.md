# Car Showroom Management System

A Java-based console application demonstrating core Object-Oriented Programming (OOP) concepts using classes, objects, encapsulation, inheritance (when applicable), and Java Collections.

##  Features
- Add new cars to the inventory  
- Sell stored vehicles  
- Search cars within configurable ranges:
  - Speed
  - Release year
  - Price
  - Engine capacity
- List all available cars in a specific range
- Clean OOP structure with separate classes for cars and managers

##  Technologies Used
- Java SE
- OOP (Classes, Objects, Encapsulation)
- Java Collections (ArrayList)
- IntelliJ IDEA

##  Project Structure

CarShowroomSystem/
 ├── src/                     # Contains all Java source code
 │    ├── cars/               # Package for car-related classes
 │    │    ├── Cars.java      # Car class (attributes, getters/setters)
 │    │    └── Main.java      # Main application with menu and user interaction & Handles inventory, add/sell/search operations
